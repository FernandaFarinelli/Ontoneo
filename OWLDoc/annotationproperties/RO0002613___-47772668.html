Annotation Property: 'axiom contradicted by evidence'

http://purl.obolibrary.org/obo/RO_0002613

Annotations (3)

  • definition "A relationship between a sentence and an instance of a piece of evidence in which the evidence contradicts the axiom" ()
  • 'curator note' "This annotation property is intended to be used in an OWL Axiom Annotation to connect an OWL Axiom to an instance of an ECO (evidence type ontology class). Because in OWL, all axiom annotations must use an Annotation Property, the value of the annotation cannot be an OWL individual, the convention is to use an IRI of the individual." ()
