Annotation Property: 'is representative IRI for equivalence set'

http://purl.obolibrary.org/obo/RO_0002617

Annotations (5)

  • 'example of usage' "In the context of a particular project, the IRI with CURIE NCBIGene:64327 (which in this example denotes a class) is considered to be representative. This means that if we have equivalent classes with IRIs OMIM:605522, ENSEMBL:ENSG00000105983, HGNC:13243 forming an equivalence set, the NCBIGene is considered the representative member IRI. Depending on the policies of the project, the classes may be merged, or the NCBIGene IRI may be chosen as the default in a user interface context." ()
  • definition "this property relates an IRI to the xsd boolean value "True" if the IRI is intended to be the representative IRI for a collection of classes that are mutually equivalent." ()
  • 'curator note' "If it is necessary to make the context explicit, an axiom annotation can be added to the annotation assertion" ()
