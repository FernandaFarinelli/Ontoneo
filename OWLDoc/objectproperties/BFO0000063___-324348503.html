Object Property: precedes

http://purl.obolibrary.org/obo/BFO_0000063

Annotations (3)

  • definition " X precedes y if and only if the time point at which x ends is before or equivalent to the time point at which y starts. Formally: x precedes y iff ω(x) <= α(y), where α is a function that maps a process to a start point, and ω is a function that maps a process to an end point." (en)
