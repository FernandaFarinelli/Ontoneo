Object Property: 'preceded by'

http://purl.obolibrary.org/obo/BFO_0000062

Annotations (3)

  • definition "X is preceded by y if and only if the time point at which y ends is before or equivalent to the time point at which x starts. Formally: x preceded by y iff ω(y) <= α(x), where α is a function that maps a process to a start point, and ω is a function that maps a process to an end point." (en)
